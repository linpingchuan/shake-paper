\begin{abstract}
\todo{\textbf{SLPJ}: Rewrite} Build systems start small and simple,
and grow into hairy monsters that few dare to touch.  One approach
this challenge, exemplified by Shake among others, is to build an
embedded domain-specfic language designed to support build systems.
We should take our own medicine: this paper describes the design of a
Shake-based build system for GHC, the Glasgow Haskell Compiler.  The
result is smaller, faster, and spectactularly more maintainable than
its \make{}-based predecessor.
\end{abstract}
